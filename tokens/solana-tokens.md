# Solana Tokens

## Solana Support In MyTonWallet

Solana is supported inside the wallet's multichain experience. Existing multichain wallets may show Solana automatically after updating.

Supported Solana features include:

- Send and receive
- SPL token balances
- Cross-chain swaps involving Solana
- NFT support
- dApp access

## SOL

SOL is Solana's native token. It is used to:

- Pay transaction fees
- Fund on-chain account rent

## SPL Tokens

SPL is the token standard on Solana. Common examples include:

- **USDT**
- **USDC**

## Rent And Token Accounts

Each SPL token usually requires its own token account on-chain. Creating that account costs a small amount of SOL in rent.

When sending a token to someone who has never held it before, that setup cost may affect the total transaction fee.

## Import Notes

Wallet import support includes wallets compatible with the **Phantom** format.

## Fees

Solana fees are very low, but you still need a small SOL balance to perform actions on Solana.
