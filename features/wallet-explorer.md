# Wallet Explorer & View Mode

## What It Is

MyTonWallet can open wallets in a read-only mode so you can inspect balances, activity, NFTs, and other public blockchain data without controlling the private keys.

This feature is also described as **View Mode**, **Watch Account**, or the **wallet explorer**.

## What You Can Do

- View any supported public wallet address
- Track balances and activity without importing the wallet
- Save watched wallets in your wallet list or gallery
- Open wallets from transaction details or pasted addresses
- Share wallet links with `my.tt` URLs

## Supported Chains

View-only and explorer support covers TON and TRON, and shared links open multichain explorer views.

## Opening A Wallet

You can usually enter view-only mode in one of these ways:

1. Add a **Watch Account** from the account switcher or wallet gallery
2. Paste an address into the explorer or search field
3. Open a shared `my.tt` wallet, transfer, or NFT link

If the app is installed, those links can open inside MyTonWallet. If it is not installed, compatible links may open in the web explorer instead.

## Watchlist And Wallet Gallery

When viewing an external wallet, you can tap **+ View** or a similar action to keep it in your watchlist. Watched wallets appear alongside your regular wallets in the wallet gallery, but remain read-only.

## Shared Links

MyTonWallet uses `my.tt` links for cleaner sharing. These links can open:

- Wallet views
- Individual transfers
- NFTs

## Explorer Options

The built-in explorer is meant for quick viewing inside the wallet. You can also switch to external explorers such as **Tonscan** or **Tonviewer** for deeper blockchain details.

## Important Notes

- View-only wallets do **not** grant spending access
- Anyone with the address can see the same public blockchain information
- Labels like known exchanges or saved contacts help with readability, but you should still verify addresses carefully
