# MyTonWallet Pay

## What It Is

MyTonWallet Pay is a wallet-payment feature for paying with crypto in supported online stores.

## Current Scope

- It was introduced around `v3.3`
- The feature is framed as a crypto checkout or payment option for merchants and stores
- Availability depends on the store integration

## Important Note

This feature currently has limited KB detail. If a user asks how to pay in a specific store, check whether that store explicitly supports MyTonWallet Pay in the current product flow.
