# TON DNS Domains

## What Are TON DNS Domains?

TON DNS domains are human-readable names that can point to a wallet address. They are NFTs on the TON blockchain, so they appear in your collectible gallery and can be transferred like other NFTs.

The most common example is a `.ton` domain such as `yourname.ton`.

## Supported Name Types

MyTonWallet also supports additional linked names, including:

- `.t.me`
- `.vip`
- `.gram`

Support can depend on the specific naming system.

## Linking A Domain

To point a domain to your wallet:

1. Open the domain from your NFT gallery
2. Choose **Link Wallet** or the equivalent action
3. Confirm the transaction

Once linked, people can send funds to the name instead of the raw address.

## Renewal And Expiry

Domains expire if they are not renewed.

The wallet provides:

- Expiry reminders inside the wallet
- Bulk renewal
- A warning before a domain expires

If a domain fully expires, someone else may be able to register it.

## Domain Management

The wallet helps you:

- View expiry
- Renew one or more domains
- Link or relink the wallet target
- Send funds to a linked domain owner

## Important Notes

- Domains are still blockchain assets, so all changes require on-chain transactions
- Not every wallet or service supports every TON-based naming format
- Domain lookalikes can be used in scams, so always verify the exact name
