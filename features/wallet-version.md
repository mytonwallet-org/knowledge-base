# Wallet Version

## What Are Wallet Versions?

On the TON blockchain, your wallet is a smart contract. Different wallet versions add different capabilities. Your wallet version determines your TON address and what features are available.

## Available Versions

- **v3R2** — widely supported, compatible with most services
- **v4R2** — adds plugin support
- **W5** — latest version, supports **gasless transactions** (paying fees in tokens like USDT instead of TON)

## Checking Your Version

Go to **Settings → Wallet Version** to see which version your wallet currently uses.

## Upgrading

You can upgrade to a newer wallet version from Settings → Wallet Version:
1. Select the version you want to upgrade to.
2. The app creates a new wallet contract with the new version.
3. Transfer your funds from the old address to the new one.

**Important:** Upgrading changes your TON address. Make sure to:
- Update your address anywhere you've shared it (exchanges, contacts, etc.)
- Transfer all tokens and NFTs from the old address to the new one

## Which Version Should I Use?

- **W5** is recommended for most users — it supports gasless transactions (see `features/gasless.md`), letting you pay fees in tokens like USDT
- **v4R2** if you need maximum compatibility with older services
- The app suggests the best version during wallet creation

## Advanced Notes

There are also more advanced wallet-version tools for power users, including TON-only wallet flows and importing older wallet versions through developer-oriented options. These are niche tools and may be hidden from normal use.
