# Portfolio Analytics

## What It Is

Portfolio is the wallet's analytics view for tracking total holdings across supported chains.

## Main Views

The wallet includes two related chart views:

- **Net Worth Chart** for a specific token or position, showing the historical value of the amount you actually held over time
- **Portfolio Chart** for the whole wallet, showing total portfolio value in your selected base currency

## What You Can See

- Combined holdings across supported chains
- Historical value changes
- Asset breakdown
- Filtering by token combination or selected assets
- Date-based inspection on the chart

## Supported Chains

The portfolio view is designed for the wallet's multichain setup, including Solana support in multichain wallets.

## Notes

- Portfolio is read-only analytics
- Displayed fiat values depend on your selected base currency
- Historical charts are informational and can vary with price-source availability
