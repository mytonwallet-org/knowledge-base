# Saved Address Book

## What Is It?

You can save frequently used wallet addresses with custom names. When sending tokens, saved addresses appear for quick selection — no need to paste or type the full address every time.

## Saving An Address

After sending to an address, the app offers to save it. You can also add addresses manually:

1. During a send flow, enter the recipient address
2. After confirming, choose **Save Address**
3. Give it a recognizable name

## Using Saved Addresses

When starting a new transfer:

1. Tap the address field
2. Your saved addresses appear as suggestions
3. Tap one to fill in the address automatically

Each saved address is associated with a specific chain (TON, TRON, or Solana), so only relevant addresses appear when sending on a particular chain.

Send flows show smarter suggestions, including:

- Saved address names
- Known-address labels for some major services or entities
- Saved addresses in Air

## Managing Saved Addresses

- To **delete** a saved address, find it in the address book and remove it.
- Saved addresses are stored locally on your device.
