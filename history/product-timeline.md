# Product Timeline

## What This File Is For

This file keeps **dated release-note facts** queryable without mixing every historical claim into the main topical help docs.

Use topical files for how features work. Use this file for **when** they were introduced or highlighted.

## Timeline

### `v3.1`

- Added connected dApps shortcuts in Explore
- Added auto-lock and password change
- Added Ledger support on mobile
- Added anti-poisoning scam labels
- Highlighted easier gasless transfers when TON balance is below `0.3 TON`

### `v3.2`

- Introduced a more visible swap aggregator with route comparison and a `Best Rate` badge
- Added manual lock
- Added push notifications for up to `3` wallets
- Added MyTonWallet Cards personalization

### `v3.3`

- Improved fee transparency with detailed fee and change breakdowns
- Expanded Explore into a dedicated tab with categories
- Introduced MyTonWallet Pay

### `v3.4`

- Launched the Telegram Mini App
- Added the Help Center to the product
- Added faster transaction actions such as copy, save address, and open in explorer

### `v3.5`

- Improved Activity feed grouping and token icons
- Added one-tap balance hiding
- Added deposit links with token selection

### `v3.6`

- Added view mode for any TON or TRON address
- Added instant transfer links after sending
- Added stronger lock-screen and passcode options

### `v3.7`

- Added multisend
- Added quick token actions
- Added advanced TON DNS management and bulk renewal
- Added pinned NFT collections
- Highlighted USDe staking

### `v3.8`

- Added richer NFT attributes viewer
- Expanded Activity feed with fiat amounts in base currency
- Added TON Connect data-signing support

### `v4.0`

- Highlighted faster incoming transfer visibility
- Added fiat amount entry for send, stake, and unstake flows
- Improved dApp interaction flow and multi-device sessions
- Exposed more advanced wallet-version tools for TON-only wallets

### `v4.1`

- Introduced MyTonWallet Air
- Added Air-specific collectibles tools, chart controls, and browser minimization
- Added new scam warnings for suspicious multisig imports and `.ton` destinations

### `v4.2`

- Added home-screen widgets and more Air improvements
- Expanded swap routing and token coverage
- Improved onboarding and Ledger support
- Added provider selection between MoonPay and Avanchange for card purchases

### `v4.3`

- Added Wallet Gallery in the classic interface
- Added balance previews and more wallet personalization
- Added Liquid Glass styling on iOS 26 in Air

### `v4.4`

- Added MoonPay off-ramp to bank cards in supported regions
- Added support for more domain types such as `.t.me`
- Added the integrated wallet explorer and `my.tt` wallet sharing
- Added smoother wallet switching in Air

### `v4.5`

- Added a token-level net worth chart
- Brought explorer and more wallet-card metadata into Air
- Added more interface languages

### `v4.6`

- Added the universal multichain web explorer
- Added iOS Lock Screen widgets
- Added saved addresses to Air
- Added explorer switching with Tonviewer support

### `v4.7`

- Added Solana support
- Added the portfolio chart
- Added pinned tokens
- Added an iPad-optimized interface
- Described Air as having full classic feature parity
