# MyTonWallet Air

## What It Is

MyTonWallet Air is the native mobile interface for iOS and Android.

## Air Highlights

Air includes:

- Faster launch and smoother native interactions
- Gesture-based navigation and wallet switching
- A minimizable built-in browser
- Advanced NFT and collectibles management
- Precise chart controls
- Wallet explorer support
- Saved addresses
- Wallet gallery and personalization
- Full classic-interface feature parity

## Core Experience

Air includes:

- Quick wallet switching
- Advanced collectibles tools
- Chart controls
- The integrated wallet explorer
- Quick token actions
- Bank-card off-ramp support
- Multisend
- Known-address labels
- Stronger NFT management

## When To Use It

Use Air as the main native mobile experience. If a guide mentions the classic interface, the same feature may appear in a different place in Air.

## Important Notes

- Air is a platform experience, not a separate wallet type
- If your interface looks different from an older guide or screenshot, check whether you are using Air or the classic mobile layout
